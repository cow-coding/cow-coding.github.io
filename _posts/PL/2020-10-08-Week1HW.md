# 1주차 과제
Q.  if문 X, else문 X, 출력구문 1개만, : 사용금지, break 사용금지, 캐스팅 X
    위 조건을 최대한 만족시키면서 아래 코드와 같은 결과를 내는 코드를 Stream, 람다식등을 써서 작성하시오.

JDK 14사용

```java
public class Hw{
  public static void main(String[] args) {
    Scanner scv = new Scanner(System.in);
		System.out.println("점수 입력 : ");
		int score = scv.nextInt();


    String grade = switch(score/10) {
		case 10, 9 -> "A";
		case 8 -> "B";
		case 7 -> "C";
		case 6 -> "D";
		default -> "F";
		};

		System.out.println(grade);
  }

}
```

정답은 JDK8 기준으로 작성을 해야했다. 그래서 어쩔 수 없이 자바 스트림과 옵셔널 공부를 했다.

맨 처음에는 한줄로 작성을 해야하나 했으나 불가능하다고 판단해서 Hash Map으로 진행을 했다.

```java
public class Hw{
  public static void main(String[] args) {
    Scanner scv = new Scanner(System.in);
		System.out.println("점수 입력 : ");
		int score = scv.nextInt();

    Map<String, Integer> grade = new HashMap<>();
		grade.put("A", 10);
		grade.put("A", 9);
		grade.put("B", 8);
		grade.put("C", 7);
		grade.put("D", 6);
		grade.put("F", 5);

    grade.entrySet().stream().filter(el -> (score / 10) <= el.getValue())
				.sorted(Map.Entry.comparingByValue())
				.findFirst().stream().forEach(el -> System.out.println(el.getKey()));

  }

}
```

우선 Map에 저장된 모든 값을 출력시키는 enrtySet을 활용해서 stream에 넣어줬다.
그 후 람다식을 활용해서 filter 진행 후 조건에 맞춰서 진행을 시켜줬다.  
가장 어려웠던 부분은 findFirst를 사용하는 걸 인지하는 것이었다.
