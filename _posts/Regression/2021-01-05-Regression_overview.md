# 회귀모델이란?

\\[
Y = \beta_{0} + \beta_{1}M + \beta_{2}F ... + \beta_{p}M^{2} + \epsilon
\\]

위의 식을 회귀모델이라고 부른다. 회귀모델은 변수의 특징과 종류에 따라 이름과 분석, 표현방식이 달라진다.   
가장 기본적인 분류로는 변수가 모두 1차식으로 이루어져있는 **선형** 회귀모델과 그렇지 않은 변수가 하나라도 존재하는 **비선형** 회귀모델이 있다.  

## 변수의 종류에 따른 회귀 모델 해석
### 선형모델과 비선형모델

아주 간단하게 생각을 해보자. 회귀분석을 시작하기 전에 두가지 모델이 주어져 있다.
1. \\(Y = \beta_{0} + \beta_{1}X + \epsilon\\)  
2. \\(Y = \beta_{0} + \beta_{1}X + \beta_{2}X^2 + \epsilon\\)  

이 두 모델의 회귀분석 결과는 당연히 다르게 나올 것이다. ($$\beta_{2}$$가 0이 아니라는 가정하에...) 그리고 당연히 최종 그래프도 다를 것이다. 만약 두 모형의 회귀분석이 끝나서 적합한 모델식을 찾았다고 가정하자. 오차항 분석은 따로 다루게 되므로 여기서는 생략한다.  

1. \\(Y = 3 + 2X$$\\)  
2. \\(Y = -7 + 2X + \frac{1}{2}X^2\\)  

그리고 이 두 식의 그래프는 아래와 같이 나오게 된다.  

![center](/image/regression/regression1.png)

예상처럼 결과도 그래프가 엄연히 다르게 나타난다. 일반적으로 파란색 그래프처럼 나오는 모델을 선형모델, 빨간 그래프처럼 나오는 모델을 비선형 모델이라한다. (물론 모든 비선형모델이 단순 이차식처럼 그래프가 그려지는 것은 아니다.)  


### 범주형 변수와 연속형 변수

우리가 사용하는 데이터에는 온도, 몸무게처럼 정확한 **수치**로 나타나는 데이터도 있지만, 성별, 인종, 혈액형처럼 숫자가 아닌 어떤 **범주**로 나타나는 데이터도 있다. 이런 데이터들을 **범주형 변수**라고 한다. 실제로 범주형 변수가 데이터로 주어지는 경우 머신 러닝, 데이터 분석에서는 해당 데이터를 수치화시켜서 분석한다. 당연히 회귀분석에서도 해당 값들을 수치화시켜서 분석한다. 아래와 같은 데이터가 들어왔을 때, 급여와 변수간의 관계를 알아보는 회귀분석을 해본다고 해보자.

|experience|sex|age|
|:---:|:---:|:---:|
|3|male|22|
|3|female|35|
|0|female|25|
|2|male|41|
|10|male|23|
|7|female|22|
|8|female|37|
|4|male|20|

위의 데이털 회귀모델을 구성한다면 아래와 같이 만들 수 있다.  

\\[
Y = \beta_{0} + \beta_{1}E + \beta_{2}S + \beta_{3}A + \epsilon
\\]  

여기서 E와 A는 수치로 나타나는 변수지만 S는 성별이므로 수치로 나타나는 변수가 아니다. 이런 경우에는 상황에 따라 데이터에 수치를 부여한다. 일반적으로 어떤 데이터 값인 경우에 높은 결과를 보이는 경우 해당 값을 회귀계수에따라 높은 수치 낮은 수치를 부여한다. 성별과 같은 수치에 다른 의미가 없는 경우에는 명목형 변수라고 부른다. 여기서는 male은 1, female은 0을 부여해서 회귀분석 값을 찾아낼 수 있다.
